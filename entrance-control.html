EntranceControl

Lightweight Security Plugin

Overview

EntranceControl is a plugin that empowers players to manage various entrances, including doors, trapdoors, and fence gates, with ease. Using a straightforward command interface, players can effortlessly select which entrances to protect and synchronize.


The plugin features an interactive and fully customizable menu, allowing players to easily manage their selected entrances. Through this menu, users can invite friends to access these entrances, facilitating collaborative gameplay.
