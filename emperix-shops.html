EmperixShops

Player Shops with Villager Trading UI

Overview

EmperixShops adds player shops where items can be bought/sold using the configured currency. All of this through the vanilla villager trade UI.


Administrators have the added benefit of creating shops with unlimited stock, utilizing the same trading mechanics. Furthermore, these shops can be designated as 'remote,' enabling players to access them via commands rather than requiring physical visits.
