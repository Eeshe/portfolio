Vendors

Immersive Villager Admin Shops

Overview

Vendors is a plugin designed for server administrators, enabling the creation of fully customizable NPC-based shops where players can sell items. Each vendor can be tailored to fit specific themes or categories, providing staff with limitless possibilities for modular shop designs.


The plugin integrates with Vault and any economy plugin to manage the currency obtained from selling items to the vendors.
